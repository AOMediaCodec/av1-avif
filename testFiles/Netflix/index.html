Netflix AVIF content

This folder contains files contributed by Netflix. The associate license is contained in license.txt.

Additional AVIF Image files are located at http://download.opencontent.netflix.com/?prefix=AV1/Chimera/AVIF/

AVIF Image files

A subfolder called avif contains files generated with libavif. These files contain HDR content encoded with different parameters (subsampling, bit depth, transfer function, bitrate). Command lines used to generate the files are in the readme.

  • HDR images

    The PNG images shown correspond to RGB data as decoded but without color profile interpretation.

    • AVIF Image Sequence files

    • Chimera-AV1-10bit-480x270.avif
      This file contains one 'pict' track and is compatible with the brands 'avis', 'msf1', 'mif1', 'miaf' and 'MA1B'.
    • alpha_video.avif
      This file contains 2 tracks: a track of type 'pict' containing a master video and a track of type 'auxv' containing an auxiliary video of type 'alpha'. Track reference and auxiliary type info box are used.
